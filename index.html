Chatwork AI Reply プライバシーポリシー

最終更新日: 2026-05-07

本拡張機能「Chatwork AI Reply」(以下、本拡張) のプライバシーポリシーです。本拡張をインストール・利用された時点で、本ポリシーに同意したものとみなします。

1. 取得・利用するデータ

本拡張は、以下のデータを取り扱います。

(1) ユーザーが自身で入力する設定値

  • Anthropic の Claude API キー
  • 使用する Claude モデルの選択値
  • コンテキストとして参照する直近メッセージの件数

(2) 返信生成のために使用するチャットデータ

  • 開いている Chatwork チャットルームの直近メッセージ本文
  • 入力欄に書かれている指示文 (スラッシュコマンドおよびユーザーが追記した指示)

2. データの保存

上記 (1) の設定値は、ブラウザの chrome.storage.local 機能を使い、ユーザーご自身の Chrome プロファイル内にのみローカル保存されます。本拡張の開発者を含む第三者がこれらのデータにアクセスすることはありません。

上記 (2) のチャットデータは、メモリ上で組み立てられ API リクエストに使用されるのみで、ローカルへの永続保存は行いません。

3. データの送信先

返信文生成リクエストにあたり、上記 (1) の API キー、(2) のチャットデータおよび指示文を、Anthropic, PBC が提供する Claude API (https://api.anthropic.com) に送信します。これらのデータを Anthropic 以外の第三者サーバー、開発者のサーバー、解析サービスなどに送信することはありません。

Anthropic 側でのデータの取扱いについては、Anthropic のプライバシーポリシー をご確認ください。

4. データの第三者提供・販売

本拡張は、ユーザーデータを第三者に販売、貸与、譲渡することはありません。また、信用力判定や融資判定の目的で使用または転送することもありません。本拡張の単一目的 (Chatwork での返信文生成支援) 以外の用途でユーザーデータを使用することはありません。

5. 自動送信について

本拡張は、生成された返信文を入力欄に下書きとして差し込むのみで、Chatwork へのメッセージ送信を自動で行うことはありません。送信操作は常にユーザーご自身の判断によって行われます。

6. ユーザーによるデータ削除

chrome.storage.local に保存された設定値は、Chrome の拡張機能管理画面から本拡張をアンインストールすることで削除されます。API キーのみを削除したい場合は、本拡張のポップアップ画面から該当フィールドを空にして保存してください。

7. お問い合わせ

本ポリシーや本拡張機能に関するお問い合わせは、Chrome Web Store のサポートタブまたは公開時に記載するメールアドレスまでご連絡ください。

8. 本ポリシーの変更

本ポリシーは予告なく変更される場合があります。変更後は本ページの最終更新日が更新されます。
